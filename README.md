\# Hospital Management Information System (HMIS)



\## 📌 Overview

A web-based system designed to manage hospital operations including patient records, doctor management, consultations, prescriptions, and reporting.



\## 🚀 Features

\- Patient registration and management

\- Doctor registration and role-based access

\- Consultation tracking per doctor

\- Prescription support with medical details

\- Invoice generation and printing

\- Secure role-based dashboards



\## 🛠️ Technologies Used

\- PHP

\- MySQL

\- HTML, CSS, JavaScript

\- XAMPP (Local development environment)



\## 📊 Project Purpose

This system was built as part of a software development project to simulate real-world hospital workflows and improve digital record keeping.
